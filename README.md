# eslintc

My Eslint Configuration

## :package: Installation

Install package

```bash
npm i -D @giovannicardamone/eslintrc
```

Add eslintrc configuration in your `package.json`

```json
{
  "name": "your-beautiful-package",
  "eslintConfig": {
    "extends": "@giovannicardamone/eslint-config"
  }
}
```
